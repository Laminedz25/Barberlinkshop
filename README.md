# BarberLink 💈
BarberLink is a premium, AI-powered platform designed to connect barbers and customers across Algeria. It provides a seamless booking experience, real-time communication, and intelligent business management for salón owners.

## 🚀 Features
- **Intelligent Booking**: Select multiple services and choose your preferred barber.
- **AI Assistant**: 24/7 AI support for customers and business analysis for barbers.
- **Barber Profiles**: Showcase your work with media galleries and manage your services.
- **Real-time Chat & Call**: Direct communication between customers and professionals.
- **Smart Notifications**: Automated booking confirmations and subscription alerts via BillionMail.
- **Interactive Map**: Discover nearby salons with ease.

## 🛠️ Technology Stack
- **Frontend**: React (Vite), TypeScript, Tailwind CSS, shadcn/ui.
- **Backend/DB**: Firebase (Firestore, Auth, Storage).
- **Email Service**: BillionMail (Self-hosted).
- **Proxy/SSL**: Traefik & Let's Encrypt.
- **Deployment**: Docker & Docker Compose on Hostinger VPS.

## 📦 Deployment
The project is containerized for easy deployment.
```bash
docker compose up -d --build
```
